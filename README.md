# vue-tabs-swiper

这是一个滑动切换的 tab 选项卡，他支持 vue/vue-jsx。

```javascript
<vue-tabs-swiper>
    <hot-spot tab-label="热点" />
    <Found tab-label="发现" />
    <Concern tab-label="关注" />
</vue-tabs-swiper>
```

## vue-tabs-swiper 参数配置

1、tab-color\<br>
2、tab-active-color\<br>
3、tab-font-weight\<br>
4、tab-font-size\<br>
