# vue-tabs-swiper

这是一个滑动切换的 tab 选项卡，他支持 vue/vue-jsx。

`npm i vue-tabs-swiper`

```javascript
<vue-tabs-swiper>
    <hot-spot tab-label="热点" />
    <Found tab-label="发现" />
    <Concern tab-label="关注" />
</vue-tabs-swiper>
```
